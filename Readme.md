# my_explorer 🚀
> 爆速で動作する、コマンドライン（CUI）専用のカスタムファイルマネージャー！

大人の事情による理不尽なブラウザ規制や、めんどくさいWindowsの環境変数設定をすべて過去にする、開発者のためのスマートなファイラーです。
専用のレトロ風インストーラーによって、再起動なしで一発で環境変数（PATH）が通り、どこからでもコマンド一発で起動できるようになります。

---

## ✨ 主な特徴
- **超軽量＆爆速動作**: 無駄なGUIを一切排除し、コマンドライン上で軽快に動作します。
- **一発セットアップ**: 配布される `Setup.exe` を実行するだけで、本体の配置、環境変数（PATH）の設定、OSへの即時反映までが全自動で行われます。
- **鉄壁のライセンス**: 著作者人格権を保護し、免責事項を網羅した安全設計。

---

## 🛠️ 使い方
権限問題を完全になくすため、**ターミナル（コマンドプロンプトやPowerShell）を「管理者権限」で開きます。**

以下のコマンドのように、プログラム名の後ろに開きたい場所の絶対パスを指定することで、そのディレクトリを即座に操作できます。

```bash
my_explorer "自分の開きたい絶対パス"
